# SuperT 初步使用说明

本文档基于当前默认编译配置整理，适用于当前工程默认的 `T5AI_BOARD` 版本。

当前默认配置特征：

- 语音输入输出已启用
- 屏幕 UI 已启用
- 当前默认对话模式为自由对话模式
- 当前默认音量为 70
- 当前默认板级按键为：
  - 音频唤醒键：`P19`
  - 网络/功能键：`P18`

## 1. 开机与首次使用

1. 给设备上电，等待系统启动完成。
2. 首次启动后，设备会初始化网络、语音和界面。
3. 如果设备尚未配网，启动后一段时间内会进入配网提示状态。

说明：

- 当前固件支持屏幕界面提示和语音提示。
- 如果设备已经成功联网，后续再次上电会自动尝试连回已保存的网络。

## 2. 配置网络

当前固件侧已启用以下配网能力：

- 蓝牙配网
- AP 配网
- 二维码配网
- FFC 一键快连

建议使用方式：

1. 打开涂鸦 App。
2. 在 App 中选择添加设备。
3. 按照 App 引导完成配网。

注意事项：

- 建议使用 `2.4G Wi-Fi`，当前能力配置未启用 `5G Wi-Fi`。
- 如果路由器名称或密码包含特殊字符，建议先使用常规英文/数字密码测试。
- 若首次配网失败，建议让设备重新进入配网状态后再试。

## 3. 按键功能

### 3.1 音频唤醒键 `P19`

当前默认固件下：

- 短按：唤醒并开始语音交互
- 长按：同样会触发唤醒并进入语音交互
- 双击：停止当前播放，并尝试切换对话模式

补充说明：

- 当前默认编译只启用了 `FREE` 自由对话模式，因此“双击切换模式”这条逻辑虽然保留，但默认版本下通常不会切换到更多模式。

### 3.2 网络/功能键 `P18`

- 短按：音量增加，按 20 为步进调整
- 双击：音量减小，按 20 为步进调整
- 长按 5 秒：清除当前网络配置并重新进入配网

## 4. 手机端可设置/可查看内容

当前源码中已确认的内容如下：

- 可设置：
  - 音量

- 可上报查看：
  - 唤醒状态
  - 联网成功状态
  - 空闲状态
  - 电量百分比（带电池板型）
  - 充电状态（带电池板型）

说明：

- 当前对象 DP 处理里，已确认直接支持的设置项主要是音量。
- 如果是带电池的板型，App 端还可以看到电量和充电状态变化。

## 5. 语音交互说明

当前默认模式为自由对话模式，典型体验如下：

1. 按下音频唤醒键，设备进入监听状态。
2. 说出问题或指令。
3. 设备上传语音并返回结果。
4. 在空闲一段时间后，设备会退出当前活跃状态。

## 6. 低功耗与唤醒

- 空闲约 30 秒后，系统会进入空闲状态
- 更长时间无活动后，系统会进入低功耗/深睡流程
- 深睡后，可通过音频唤醒键再次唤醒设备

说明：

- 当前版本中，音频唤醒键同时承担运行态交互和深睡唤醒功能。

## 7. 常见问题

### 7.1 配网失败怎么办？

建议依次检查：

- 手机是否连接在 `2.4G Wi-Fi`
- 路由器网络是否正常
- 设备是否确实进入配网状态
- 是否长按网络键 5 秒重新清网再配

### 7.2 语音键按下没有反应怎么办？

建议检查：

- 当前固件是否确实烧录为默认 `T5AI_BOARD` 版本
- `P19` 按键硬件电平是否正常
- 麦克风、喇叭、电源是否工作正常

### 7.3 音量变化为什么不是一格一格变化？

因为当前本地按键音量调整逻辑使用的是 `20` 为步进。
